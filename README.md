# Stack base

1. MongoDB 5.0.
2. Nest JS.

# Ejecutar el desarrollo

1. Clonar el repositorio.
2. Ejecutar:

```
npm install
```

3. Tener Nest CLI instalado:

```
npm install -g @nestjs/cli
```

4. Levantar la base de datos Mongo:

```
docker-compose up -d
```
